## What is GIWA?

GIWA chain is an Ethereum Layer2 solution built on OP Stack, a next-generation blockchain with low gas fees and high-performance transaction processing capabilities. Based on high EVM compatibility, you can easily utilize smart contracts and DApps from the existing Ethereum ecosystem, providing a developer-friendly and scalable environment.

GIWA Sepolia is the testnet of the GIWA network, where you can freely test smart contract deployment, transaction processing, etc. without using real assets.

GIWA Wallet can be created through EVM-compatible wallets such as Metamask, just like general EVM chain wallet creation methods, and can be connected by setting the network to GIWA Sepolia.

For those connecting a wallet for the first time, please refer to the [official guide](https://docs.giwa.io/get-started/connect-to-giwa)!
