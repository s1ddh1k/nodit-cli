## What is Kaia And Kairos?

Kaia is an integrated blockchain network brand of Klaytn and Finschia, a mainnet operated by hard forking the Klaytn network. Since it is based on Klaytn, it is compatible with EVM and can be easily used by users who were using Klaytn.

Kairos is Kaia's testnet configured by hard forking the Klaytn Baobab network, which is the testnet of the Klaytn network.
