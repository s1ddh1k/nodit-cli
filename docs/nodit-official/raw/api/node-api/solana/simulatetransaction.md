# simulateTransaction

**`POST /`**

Simulate sending a transaction

## Request Body

| Name | Type | Required | Description |
|------|------|----------|-------------|
| jsonrpc | string | ✓ |  |
| id | integer | ✓ |  |
| method | string | ✓ |  |
| params | array | ✓ | It contains following items:   1. Transaction (`required`)   2. ConfigurationObject (`optional`)  |

## Response

| Name | Type | Required | Description |
|------|------|----------|-------------|
| id | integer |  | The request ID |
| jsonrpc | string |  | The JSON-RPC version |
| result | object |  |  |
| result.context | object |  | An object containing metadata about the current state of the Solana network at the time of the request. |
| result.context.apiVersion | string |  | The API version used for the request |
| result.context.slot | integer |  | The slot number at which the request was evaluated |
| result.value | object |  |  |
| result.value.accounts | array |  |  |
| result.value.err | object |  | Error if transaction failed, null if transaction succeeded. |
| result.value.innerInstructions | array |  | Defined only if innerInstructions was set to true. The value is a list of inner instructions. |
| result.value.loadedAccountsDataSize | integer |  | The number of bytes of all accounts loaded by this transaction |
| result.value.logs | array |  | Array of log messages the transaction instructions output during execution, null if simulation failed before the transaction was able to execute |
| result.value.replacementBlockhash | object |  |  |
| result.value.returnData | object |  | The most-recent return data generated by an instruction in the transaction  |
