# getTransaction

**`POST /`**

Returns transaction details for a confirmed transaction

## Request Body

| Name | Type | Required | Description |
|------|------|----------|-------------|
| jsonrpc | string | ✓ |  |
| id | integer | ✓ |  |
| method | string | ✓ |  |
| params | array | ✓ | It contains following items: 1. Transaction signature (`required`) 2. ConfigurationObject (`optional`)  |

### Example

```json
{
  "jsonrpc": "2.0",
  "id": 1,
  "method": "getTransaction",
  "params": [
    "5Pj5fCupXLUePYn18JkY8SrRaWFiUctuDTRwvUy2ML9yvkENLb1QMYbcBGcBXRrSVDjp7RjUwk9a3rLC6gpvtYpZ",
    {
      "commitment": "confirmed",
      "maxSupportedTransactionVersion": 0,
      "encoding": "json"
    }
  ]
}
```

## Response

| Name | Type | Required | Description |
|------|------|----------|-------------|
| id | integer |  | The request ID |
| jsonrpc | string |  | The JSON-RPC version |
| result | object |  |  |
| result.blockTime | integer |  | Estimated production time, as Unix timestamp (seconds since the Unix epoch) of when the transaction was processed. null if not available |
| result.meta | object |  |  |
| result.meta.err | object |  | Error if transaction failed, null if transaction succeeded. |
| result.meta.fee | integer |  | fee this transaction was charged, as u64 integer |

| result.meta.postBalances | array |  | array of u64 account balances after the transaction was processed |
| result.meta.innerInstructions | array |  | List of inner instructions or null if inner instruction recording was not enabled during this transaction |
| result.meta.preTokenBalances | array |  | List of token balances from before the transaction was processed or omitted if token balance recording was not yet enabled during this transaction |
| result.meta.postTokenBalances | array |  | List of token balances from after the transaction was processed or omitted if token balance recording was not yet enabled during this transaction |
| result.meta.logMessages | array |  | array of string log messages or null if log message recording was not enabled during this transaction |
| result.meta.rewards | array |  | transaction-level rewards  |

| result.meta.returnData | object |  | the most-recent return data generated by an instruction in the transaction, with the following fields:  |

| result.meta.version | string |  |  |

| result.transaction | object |  |  |
