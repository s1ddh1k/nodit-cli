# klay_getFilterChanges

**`POST /`**

Returns filter results by filter ID. Enter the filter ID generated by klay_newFilter, klay_newBlockFilter, or klay_newPendingTransactionFilter.

## Request Body

| Name | Type | Required | Description |
|------|------|----------|-------------|
| id | integer | ✓ |  |
| jsonrpc | string | ✓ |  |
| method | string | ✓ |  |
| params | array | ✓ | Enter the following parameters as an array with the appropriate types. 1. `filter ID`: Enter the previously created filter ID. |

### Example

```json
{
  "id": 1,
  "jsonrpc": "2.0",
  "method": "klay_getFilterChanges",
  "params": [
    "0xaf35d60b70eb3b54018456a0d365ea49"
  ]
}
```
