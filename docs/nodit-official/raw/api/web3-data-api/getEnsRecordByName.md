# Get ENS Record By Name

**`POST /{chain}/{network}/ens/getEnsRecordByName`**

Retrieves ENS domain information by entering the ENS domain name.
		

> 🚧 Please verify the network when using!
>
> This API is supported only on Ethereum Mainnet and cannot be used on other networks. Please verify the network when using.

## Parameters

| Name | In | Type | Required | Description |
|------|----|------|----------|-------------|
| chain | path | string | ✓ | Parameter to specify the target chain for the query. |
| network | path | string | ✓ | Parameter to specify the network of the target chain. Supported networks may vary depending on the chain. |

## Request Body

| Name | Type | Required | Description |
|------|------|----------|-------------|
| name | string |  | Enter the ENS domain name. |

## Response

| Name | Type | Required | Description |
|------|------|----------|-------------|
| name | string | ✓ | A field representing the name of the ENS domain. |
| isMigrated | boolean | ✓ | A field indicating whether the ENS domain has been migrated to the latest version. If true, the domain has been migrated to the latest version; if false, the domain has not been migrated. Migration status is important for domain management and compatibility. |
| isPrimaryName | boolean | ✓ | A field indicating whether the ENS domain is the Primary Name. In the ENS structure, multiple domains can be linked to a single address, and the primary domain is called the Primary Name. If true, the domain is registered as Primary Name; if false, it indicates a non-Primary domain. |
| node | string | ✓ | A field representing the unique identifier of the ENS domain, which is a hash value generated by the namehash algorithm. For example, if the domain name is example.nodit.eth, the node value is generated as namehash('example.nodit.eth'). Provided as a 64-character hexadecimal string prefixed with 0x. |
| labelName | string | ✓ | A field representing the lowest-level label of the domain name. For example, for example.nodit.eth, labelName is 'example'. |
| labelHash | string | ✓ | A field representing the hash value of the lowest-level label of the domain name. For example, for example.nodit.eth, labelHash is generated as keccak256('example'). Provided as a 64-character hexadecimal string prefixed with 0x. <br/> <strong style='color: red;'>*</strong> For wrapped domains, the address shown in ownerAddress may be the wrapping contract address. Please refer to the wrappedOwnerAddress field to verify the actual owner address. |
| ownerAddress | string | ✓ | A field representing the owner address of the ENS domain. Provided as a 40-character hexadecimal string prefixed with 0x. <br/> <strong style='color: red;'>*</strong> The owner address indicates the owner of the domain and may differ from the address the domain is linked to. For asset transfers, please refer to resolvedAddress in the resolver field. |
| resolver | object | ✓ | A field representing the Resolver information of the domain. |
| resolver.contractAddress | string | ✓ | A field representing the Resolver contract address of the domain. Provided as a 40-character hexadecimal string prefixed with 0x. |
| resolver.resolvedAddress | string | ✓ | A field representing the address the domain is linked to. Provided as a 40-character hexadecimal string prefixed with 0x. <br/> <strong style='color: red;'>*</strong> resolvedAddress may differ from the owner address. Please refer to the ownerAddress field to verify the actual owner address of the domain. |
| resolver.contentHash | string | ✓ | A field representing the content hash of the domain. Stores various forms of content hash including IPFS hashes in ERC-1577 format. This field provides the decoded result. |
| resolver.text | object | ✓ | A field representing the text record information of the domain. Text records are fields that can store various information about the domain; ENS domain owners can store any desired information. <br/> <strong style='color: red;'>*</strong> Different text records may exist for each domain. If no text record exists, this field is provided as an empty object. |
| registration | object | ✓ | A field representing the registration information of the domain. |
| registration.registrantAddress | string | ✓ | A field representing the address that registered the domain. Provided as a 40-character hexadecimal string prefixed with 0x. |
| registration.registrationDate | integer | ✓ | A field representing the registration date of the domain. Provided in ISO 8601 format (YYYY-MM-DDThh:mm:ss{time zone}). |
| registration.expiryDate | integer | ✓ | A field representing the expiry date of the domain. Provided in ISO 8601 format (YYYY-MM-DDThh:mm:ss{time zone}). |
| registration.cost | string | ✓ | A field representing the registration cost of the domain in Wei. Provided in decimal string format. |
| nameWrapper | object | ✓ | A field representing the wrapped domain information. For wrapped domains, the ownerAddress field displays the wrapping contract address. Please refer to the wrappedOwnerAddress field to verify the actual owner address. |
| nameWrapper.contractAddress | string |  | A field representing the wrapping contract address of the domain. Provided as a 40-character hexadecimal string prefixed with 0x. If the domain is not wrapped, this field is provided as null. |
| nameWrapper.wrappedOwnerAddress | string |  | A field representing the owner address of the wrapped domain. Provided as a 40-character hexadecimal string prefixed with 0x. If the domain is not wrapped, this field is provided as null. |
